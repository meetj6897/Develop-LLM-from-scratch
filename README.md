# Develop-LLM-from-scratch
This repo have process of developing the LLM from the scratch with Byte Pair Encoding, Token &amp; Positional Embeddings, Self-Attention Mechanism, Causal Self-Attention,  Multi-Head Attention,  Layer Normalization, GELU Activation,  Autoregressive Next-Token Prediction, Temperature Scaling, Top-k Sampling, Classification &amp; Instruction Fine-Tuning.
